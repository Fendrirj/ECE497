To play, run the sketch program.

First, wire up your buttons to the following ports:
14, 16, 21, 22

Remember, 14 and 16 use pull down resistors and 21 and 22 use pull up,
so wire your buttons up accordingly.

Once set up, you may run the program.

CONTROLS:
port 14 controls up
port 16 controls left
port 21 controls right
port 22 controls down

To exit, press ctrl-c
